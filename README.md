# Web-Scraping-
This project focuses on extracting real-world data from a website using web scraping techniques and performing data cleaning and exploratory data analysis using Python.
## Tools & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas
## Workflow
1. Sent HTTP requests to the target website
2. Parsed HTML content using BeautifulSoup
3. Extracted relevant data fields
4. Cleaned and structured the data into a DataFrame
5. Exported the final dataset to CSV format
6. Performed basic exploratory data analysis
## Key Learnings
- Understanding website structure and HTML tags
- Handling missing and inconsistent scraped data
- Converting raw web data into analysis-ready datasets
## Conclusion
This project demonstrates the ability to collect, clean, and analyze real-world data
from web sources, which is a crucial step in practical data science workflows.
